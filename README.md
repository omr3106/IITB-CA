# IITB-CA

A collection of Computer Architecture (CA) resources, notes, assignments, implementations, and reference materials inspired by concepts taught in Computer Architecture courses at the Indian Institute of Technology Bombay (IIT Bombay).

## 📖 Overview

This repository serves as a centralized resource for learning and practicing Computer Architecture concepts. It includes study materials, problem-solving approaches, code implementations, and other academic resources useful for students and enthusiasts.

## 🚀 Topics Covered

- Computer Organization Fundamentals
- Instruction Set Architecture (ISA)
- CPU Design and Datapath
- Pipelining
- Hazard Detection and Resolution
- Memory Hierarchy
- Cache Memory
- Virtual Memory
- Input/Output Systems
- Parallel Processing
- Performance Analysis
- Assembly Language Concepts
